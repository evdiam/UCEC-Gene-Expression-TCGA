# UCEC-Gene-Expression-TCGA
## Differential Gene Expression Analysis of Uterine Corpus Endometrial Carcinoma (TCGA-UCEC)

Aim of project:
This project aims to analyse the RNA sequencing data from the program TCGA for the disease UCEC (Uterine Corpus Endometrial Carcinoma), to investigate differences in gene expression between tumour and normal uterine tissue, using the programming languages Python and R. The workflow includes data cleaning, exploratory analysis, statistical testing, visualisation and biological interpretation.
